### **Digi-Tools Platform**
 
This platform helps a new learning or professional programmer to buy their important toolkits. The tools are displayed in a user-friendly version so that anyone can fully understand and buy a tool based on their need. <br>
The pricing is also included and the checkout can be done remaining in this platform. 
So, No longer a user has to go to different platforms for different toolkits.


### The technologies used in this app

This project is solely based on react javascript <br>
It also uses a react icons, daisyUI JSX, and tailwind CSS. Also 'Manrope' google font are used in this Project

### Features
- This projects accumulates many different AI toolkits for a user with vast diversified needs.
- The Toolkits are arranged in a user friendly way so that no one stumbles in their way of finding their preferred tool
- Any user can buy the toolkits from remaining in this browser reducing the turmoil of buying a tool from different "reliable places".
